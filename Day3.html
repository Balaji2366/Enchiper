Steps to Upload an Image

  1. Open the website or application where you want to upload the image.
  2. Click on the Upload or Add Image button.
  3. Select the image file from your computer or mobile.
  4. Click the Open button to choose the image.
  5. Finally, press the Submit or Save button to upload.

Common Image Formats

JPEG
JPEG is a compressed image format mainly used for photographs. It reduces file size but slightly loses quality.
PNG
PNG supports transparent backgrounds and provides high image quality. It is commonly used for logos and web graphics.
SVG
SVG is a vector-based image format that can be scaled to any size without losing quality. It is used mostly for icons and illustrations.

HTML Tags Used in UI Design

div
The div tag is a container used to group elements and design layouts. It is widely used for creating sections and UI components.
span
The span tag is an inline container used to style small parts of text or elements without breaking the layout.
header
The header tag is used to define the top section of a webpage such as logos, titles, and navigation menus.
